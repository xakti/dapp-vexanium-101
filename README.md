# DApp Vexanium 101
## Bangun DApp Vexanium Pertamamu Sekarang

Repositori ini adalah tempat yang sempurna bagi siapa saja yang ingin menciptakan DApp inovatif di blockchain Vexanium.

DApp ini dikembangkan sebagai Web Progressive App (PWA), sehingga dapat di-install langsung di perangkat pengguna (desktop maupun mobile) dan digunakan seperti aplikasi native.

## Apa yang akan kamu pelajari
- menghubungkan DApp ke Dompet Herlina
- transfer token VEX dengan fitur auto complete alamat penerima
- membaca riwayat transfer masuk
- tanda tangan pesan
